Public Works

Arya Kaul

There are a number of excellent essays/novels/speeches/poems in the public domain for which no well-formatted pdf exists. Without changing the source material, this project simply involves me type-setting some of these works in a cohesive manner through Latex. If you have any suggestions for pieces to include, feel free to email them to me! Please find the current completed pdfs at the following link. I'm more than happy to share raw tex files on request.
